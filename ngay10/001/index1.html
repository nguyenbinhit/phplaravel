cú pháp câu lệnh điều kiện trong js

        1-cú pháp rút gọn
        if(điều kiện) {
            //các câu lệnh sẽ chạy khi điều kiện đúng
        }

        2-cú pháp if else
        if(điều kiện) {
            //các câu lệnh sẽ chạy khi điều kiện đúng
        }else{
            //các câu lệnh sẽ chạy khi điều kiện sai
        }

        3-cú pháp if else if

        4-cú pháp rút gon
        (điều kiện) ? (câu lệnh chạy khi điều kiện đúng) : (câu lệnh chạy khi mà điều kiện sai)
        - dùng khi mà câu lệnh thực thi trong {} chỉ có một dòng duy nhất và thường đucợ dùng để gán giá trị cho một biến khác
